  • 질문1 : nav 태그에서 z-index: 999;의 기능은 무엇인가요
    답변1: z-index는 요소가 겹쳤을 때 어떤 요소가 위에 표시될지를 결정하는 CSS 속성입니다. 숫자가 클수록 더 위쪽(앞쪽)에 배치됩니다. 📌 z-index: 999;의 의미 nav 태그에 z-index: 999;를 주면, 다른 요소들과 겹칠 때 nav가 거의 항상 맨 위에 보이도록 합니다. 예를 들어, 스크롤 시 배너나 이미지, 달력 같은 요소가 nav와 겹친다면 z-index 값이 높은 nav가 위에 표시됩니다. z-index는 position 속성이 있는 요소(relative, absolute, fixed, sticky)에만 적용됩니다. 따라서 nav에 position: fixed;나 position: relative; 같은 속성 이 있어야 효과가 있습니다.
    예제]
    .navbar { position: fixed; /* 화면 상단에 고정 */
    top: 0;
    left: 0;
    width: 100%;
    z-index: 999; /* 다른 요소보다 위에 표시 */
    }
    이렇게 하면 스크롤을 내려도 nav 메뉴가 항상 화면 맨 위에 떠 있고, 다른 콘텐츠가 겹쳐도 nav가 위에 보입니다.
